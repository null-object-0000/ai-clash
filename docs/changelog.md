# 更新日志

## [1.1.0] - 2026-04-16

### 新增 (Added)
- **“沉浸跟随” (赛博导播) 模式**：新增自动化标签页切换模式。启用后，扩展会自动无缝切换标签页以跟随正在输出的 AI 通道，提供瀑布流般的观看体验，并完美绕过浏览器的后台休眠限制。
- **官方网站多语言支持**：为我们基于 Vitepress 搭建的官网引入了中英文双语支持。

### 修复 (Fixed)
- **AI 总结按钮显示逻辑**：修复了在 AI 通道仍在输出内容时，“生成总结”按钮会错误显示的问题。现在在生成中状态将自动隐藏。
- **空总结气泡**：修复了可能会渲染出空总结气泡的 Bug。加入了稳健的内容验证逻辑，确保 UI 呈现更干净利落。
