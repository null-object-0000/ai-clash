# 更新日志

## [1.2.0] - 2026-04-24

### 🎉 重磅更新 (Major)

- **接入新通道 - Xiaomi MIMO**：现在支持 Xiaomi MIMO 了，可以直接加入多通道对比。
- **单轮对话可中途加通道**：提问后不用重来，回答过程中也能随时再加一个 AI 通道一起对比。

### 🛠️ 优化与修复 (Fixes & Improvements)

- **通道列表体验优化**：支持全选/反选，按国内/海外分组展示，高度自适应侧边栏空间。
- **多通道派发稳定性优化**：重构任务调度与错误处理流程，减少网页模式下通道唤醒、页面就绪和消息派发阶段的异常。
- **多通道默认行为优化**：当启用 2 个及以上通道时，`自动总结` 和 `导播模式` 会默认开启；如果你手动改过开关，系统会记住并尊重你的选择。
- **失败通道支持单独重试**：每个回答通道现在都可以在失败后单独重试，不必整轮重新提问；历史记录中的失败通道也支持直接重试。
- **历史记录错误态恢复修复**：修复了失败通道被错误保存为完成态的问题，重新打开历史会话时会正确显示失败状态和对应操作。
- **DeepSeek 登录页快速失败**：当 DeepSeek 明确跳转到登录页时，现在会立刻报错并继续后续通道，不再长时间卡在等待状态。
- **移除元宝智能搜索手动切换**：腾讯元宝现在会根据提问自动联网，不需要你再手动开关了。

## [1.1.0] - 2026-04-17

### 🎉 重磅更新 (Major)

- **接入新通道 - 文心一言**：现在支持文心一言了，可以直接加入多通道对比。
- **「焦点跟随」导播模式**：哪个 AI 正在输出，就自动把焦点切到哪个页面，尽量减少后台页面被浏览器限速的问题。

### ✨ 新增特性 (Features)

- **自定义归纳总结**：可以在全局设置里改「AI 裁判」的提示词，让总结风格更符合你的习惯。
- **归纳交互重做**：总结改为手动触发，还加了「重新生成」，你可以自己决定什么时候总结。
- **官网与分发中心**：上线了官网，能查看历史版本，也有离线 ZIP 包和开发者模式安装说明。

### 🛠️ 优化与修复 (Fixes & Improvements)

- **追问拦截防护**：多通道模式下新增防误触弹窗，避免手滑追问把还没答完的内容清掉。

## [1.0.0] - 2026-04-15

### 🎉 创世版正式上线 (MVP Release)

- **同屏多通道并发对比**：新增独立侧边栏工作台，一次提问可以让多个网页同时回答。
- **零 API 成本架构**：不用自己申请和配置 API，直接复用各平台官网网页端能力。
- **首发四大通道**：首个版本支持 DeepSeek、豆包、通义千问、腾讯元宝。

### ✨ 核心特性 (Features)

- **内置 AI 裁判引擎**：各通道回答后会自动做一版总结，帮你快速看出共识和差异。
- **原生模式映射**：会同步各平台的常用开关，例如「深度思考」和「联网搜索」状态。
