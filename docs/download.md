# 📥 插件下载与安装

## 方式一：应用商店一键安装 (推荐)

如果你可以正常访问外网，强烈推荐通过官方商店安装，这样可以获得**自动更新**的能力。
- [Chrome Web Store 官方下载](https://chromewebstore.google.com/detail/ggngmgpjdklmkpoldbfahmeefpnfhhai)
- [Edge Add-ons 官方下载](https://microsoftedge.microsoft.com/addons/detail/khjmihaeihajagobgbdhlbjeobdpmfkm)

## 方式二：离线 ZIP 包开发者模式安装 (国内免翻)

无法访问商店？没关系，按照以下 3 步即可轻松安装。

### 1. 下载最新版 ZIP 包

👉 **[点击下载 AI 对撞机 v1.1.0 离线包](https://github.com/null-object-0000/ai-clash/releases/latest)** (跳转后下载 `ai-clash-v1.1.0.zip`)

### 2. 解压文件
将下载的 ZIP 文件解压到电脑上的一个常用文件夹中（请不要删除该文件夹）。

### 3. 在浏览器中加载

1. 打开浏览器的扩展管理页面：
   - Chrome 用户在地址栏输入：`chrome://extensions/`
   - Edge 用户在地址栏输入：`edge://extensions/`
2. 打开右上角的 **「开发者模式」** 开关。
3. 点击左上角的 **「加载已解压的扩展程序」**，然后选择你刚刚解压好的文件夹即可！🎉
