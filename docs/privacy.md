# AI 对撞机隐私政策

最后更新时间：2026年4月16日

## 1. 数据收集
AI 对撞机是一款纯正的浏览器端工具。所有 AI 模型交互均直接发生在您的浏览器和相应的 AI 服务提供商（例如 DeepSeek、Kimi 等）之间。我们**不会**收集、存储或向我们自己的服务器传输您的对话、提示词或任何个人数据。

## 2. 权限说明
- **标签页/ActiveTab (Tabs)**：用于在提供对话服务的不同 AI 标签页之间进行同步和切换，以提供无缝体验。
- **本地存储 (Storage)**：用于在本地保存您的偏好设置，例如已启用的模型和自定义的提示词。
- **脚本注入 (Scripting/Content Scripts)**：用于与您已经登录的 AI 平台网页界面进行交互（获取网页原生算力）。

## 3. 第三方服务
此扩展程序会与第三方 AI 网站进行交互。您对这些服务的使用受其各自隐私政策的约束。

## 4. 联系我们
如果您有任何问题，请通过 GitHub Issues 联系我们。
