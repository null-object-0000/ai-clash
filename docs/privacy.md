# AI 对撞机隐私政策

最后更新时间：2026年4月24日

## 1. 数据收集

AI 对撞机主要在您的浏览器本地运行。网页通道的对话通常直接发生在您的浏览器和相应的 AI 服务提供商之间，例如 DeepSeek、豆包、通义千问、腾讯元宝、文心一言和 Xiaomi MIMO。

当您使用「内置总结服务」时，AI 对撞机会将您的原始问题以及被选中通道的回答内容发送至 AI 对撞机官方总结服务 `https://ai-clash-service.snewbie.site`，用于生成归纳总结。相关内容仅用于处理本次总结请求，不会用于训练模型，也不会由我们主动共享给第三方。

如果您不希望总结内容发送至 AI 对撞机官方总结服务，可以关闭自动总结，或在设置中改用您自己配置 API Key 的总结通道。

## 2. 权限说明

- **标签页/ActiveTab (Tabs)**：用于在提供对话服务的不同 AI 标签页之间进行同步和切换，以提供无缝体验。
- **本地存储 (Storage)**：用于在本地保存您的偏好设置，例如已启用的模型和自定义的提示词。
- **脚本注入 (Scripting/Content Scripts)**：用于与您已经登录的 AI 平台网页界面进行交互（获取网页原生算力）。
- **站点权限 (Host Permissions)**：用于访问支持的 AI 网站、支持的 API 端点，以及在您使用内置总结功能时访问 AI 对撞机官方总结服务。

## 3. 第三方服务

此扩展程序会与第三方 AI 网站和 API 服务进行交互。您对这些服务的使用受其各自隐私政策的约束。
