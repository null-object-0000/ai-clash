# 📥 Download & Installation

## Method 1: Store Installation (Recommended)

If you can access the stores, it is highly recommended to install through them to get **automatic updates**.
- [Chrome Web Store](https://chromewebstore.google.com/detail/ggngmgpjdklmkpoldbfahmeefpnfhhai)
- [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/khjmihaeihajagobgbdhlbjeobdpmfkm)

## Method 2: Offline ZIP Installation (Developer Mode)

Cannot access the store? No problem, just follow these 3 steps.

### 1. Download the Latest ZIP

👉 **[Download AI Clash v1.2.0 Offline Package](https://github.com/null-object-0000/ai-clash/releases/latest)** (Download `crx-ai-clash-1.2.0.zip` after redirect)

### 2. Unzip Files

Extract the downloaded ZIP file to a folder on your computer (please do not delete this folder later).

### 3. Load in Browser

1. Open your browser's extensions management page:
   - Chrome: Type `chrome://extensions/` in the address bar.
   - Edge: Type `edge://extensions/` in the address bar.
2. Turn on the **"Developer mode"** switch in the top right corner.
3. Click **"Load unpacked"** in the top left, and select the folder you just extracted! 🎉
